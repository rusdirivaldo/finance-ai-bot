🧠 Project Summary
AI Personal Finance Telegram Bot

Project ini adalah Telegram Bot Personal Finance Assistant yang memungkinkan user mencatat transaksi keuangan menggunakan natural language, lalu bot akan:

menyimpan transaksi

mengkategorikan otomatis

membuat laporan keuangan

menganalisis kondisi finansial

menampilkan dashboard analytics

Bot di-deploy menggunakan serverless architecture.

🗄 Database Structure
Table: transactions
id
chat_id
date
type            (income | expense)
category
description
amount
needs_category

Purpose:

store all financial transactions
Table: categories
id
name
keywords
level

Example:

name	keywords	level
food	makan,warteg	primer
transport	bensin,tol	primer
shopping	tokopedia,shopee	sekunder

Level meaning:

primer   → Essential spending
sekunder → Lifestyle spending
tersier  → Luxury spending
Table: budgets
id
chat_id
category
monthly_limit
month
year

Constraint:

UNIQUE(chat_id, category, month, year)

Purpose:

prevent duplicate budgets
✨ Core Features
1️⃣ Natural Language Expense Input

User can type:

makan ramen 85k
beli oli motor 65k
ngopi starbucks 45k

Bot automatically:

parse amount
detect type
detect category
insert transaction

Amount parser:

85k → 85000
2jt → 2000000
2️⃣ Auto Category Detection

Algorithm:

tokenize input
remove stopwords
keyword match
string similarity

Example:

beli oli motor
→ transport
3️⃣ Stopword Filtering

Example stopwords:

beli
bayar
order
ambil
isi
topup

So:

beli kopi starbucks
→ keyword detected = kopi
4️⃣ Auto Category Learning

When category not detected:

category = other
needs_category = true

Bot response:

❓ Kategori belum dikenali

User command:

/setcategory transport

Bot will:

update category keywords
update transaction
